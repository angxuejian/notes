# th 家族


## 指示词（Demonstratives）

| 单词    | 意思 | 作用       |
| ----- | -- | -------- |
| this  | 这个 | 靠近我      |
| that  | 那个 | 离我远      |
| these | 这些 | this 的复数 |
| those | 那些 | that 的复数 |

## 人称代词（Pronouns）

| 单词     | 意思       |
| ------ | -------- |
| he     | 他        |
| she    | 她        |
| they   | 他们/她们/它们 |
| them   | 他们（宾格）   |
| their  | 他们的      |
| theirs | 他们的（名词性） |

## 地点（Place）

| 单词    | 意思 |
| ----- | -- |
| here  | 这里 |
| there | 那里 |
| where | 哪里 |

## 时间（Time）

| 单词   | 意思    |
| ---- | ----- |
| now  | 现在    |
| then | 那时；然后 |
| when | 什么时候  |

## 比较（Comparison）

| 单词      | 意思 |
|-----------|------|
| than      | 比；比……更 |

## 数字（Numbers）

| 单词      | 意思 |
|-----------|------|
| three     | 三 |
| third     | 第三 |
| thirteen  | 十三 |
| thirty    | 三十 |

## 常见连接词（Connectors）

| 单词        | 意思 |
|------------|------|
| that       | （引导从句，很多时候不翻译） |
| though     | 虽然 |
| although   | 尽管 |
| therefore  | 因此 |
| thus       | 因此（正式） |
